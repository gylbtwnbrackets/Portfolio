Un professionnel RT peut souvent recontrer des données sous formes brut, pour les exploiter, il doit savoir comment les traiter et les présenter, et éventuellement automatiser les processus Ce projet avait pour Objectifs de nous familiariser avec les méthodes de traitement de données

Apprentissages Critiques :

Durant ce projet, j'ai du mobiliser mes compétences en programmation dans le language python, ainsi que mes compétences en dévloppemnt web, pour créer un outil permettant de formaliser les données de Description de tous les Pokémons existant.
