Projet n°7 – SAE 2.03 : Mise en place de Services (Web/VoIP)

Objectifs du Projet

Ce projet consistait à configurer des services de communication essentiels pour une entreprise. Nous avons mis en place un serveur Web dynamique (Apache/Nginx) et une solution de Téléphonie sur IP (Asterisk).

Compétences visées

Compétence principale : Créer des outils et applications informatiques (Services Réseaux)

Apprentissages Critiques :

  • Connaître l’architecture et les technologies d’un site Web (AC13.04)
  • Choisir les mécanismes de gestion des données adaptés (AC13.05)

Déroulement du projet

J'ai été chargé de la configuration du serveur Web et de la base de données associée. J'ai aussi participé à la configuration du plan de numérotation sur le serveur VoIP pour permettre les appels internes.

Réalisations et Livrables

  • Fichiers de configuration Asterisk
  • Lien vers le site web déployé

Réflexion et Autoévaluation

Points forts : La mise en place du serveur Web a été rapide et fonctionnelle.

Difficultés rencontrées : La configuration du NAT pour la VoIP a posé problème (problème de son à sens unique).
