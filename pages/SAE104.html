Projet n°4 – Se présenter sur Internet

Objectifs du Projet

L'objectif de cette SAE était de créer notre propre présence numérique sous la forme d'un e-portfolio. Nous devions utiliser les langages HTML et CSS pour structurer et styliser notre site, en veillant à l'identité numérique et au référencement.

Compétences visées et Apprentissages Critiques

Compétence principale : Communiquer professionnellement sur le web.

Apprentissages Critiques :

  • Maîtriser les bases du développement web (HTML5, CSS3)
  • Gérer son identité numérique
  • Utiliser des outils de versionning comme Git et GitHub

Déroulement du projet

J'ai commencé par maquetter le site pour définir la structure. Ensuite, j'ai codé les différentes pages en essayant de maintenir un design cohérent. J'ai appris à héberger mon code sur un dépôt GitHub.
